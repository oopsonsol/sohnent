# **App Name**: Sohn Enterprises

## Core Features:

- Home Landing Page: Display a full-screen hero section with the company wordmark, subheading, and a discreet CTA.
- Capital Deployment Section: Show a minimal section with a brief description of capital deployment activities.
- Asset Acquisition Section: Show a minimal section describing strategic ownership positions.
- Development & Holdings Section: Show a minimal section outlining structured execution.
- Regulatory Alignment Section: Show a minimal section emphasizing operations consistent with U.S. governance standards.
- About Page: Display a minimal 'About' page with a headline and four ultra-short lines about the company.
- Contact Form: Implement a static contact form with name, email, and message fields that posts to an external service (Netlify Forms, Formspree, etc.).

## Style Guidelines:

- Primary color: Off-white (#F5F5F5) for a clean and institutional feel.
- Background color: Charcoal (#333333) for a powerful, discreet base.
- Accent color: Subtle gold (#D4AF37) for highlights and an expensive feel.
- Font: 'Inter', a modern sans-serif, for all headlines and body text to ensure tall, clean, and professional readability.
- Brutalist spacing to convey a sense of intentional design and premium feel, inspired by BlackRock and Apollo.
- Organic fade/opacity animations using Tailwind + CSS for subtle transitions.
